A widget to show the latest post with the most commented post, including a customizable number of posts and excerpt length.

== Description ==

The Simple Most Commented Post plugin allows you to display the most commented posts in your sidebar or any widget area. It is highly customizable, allowing you to set the number of posts to display, the length of the post excerpt, and the widget title.

**Features:**
* Display the most commented posts.
* Customize the number of posts shown.
* Set the length of the post excerpt.
* Easily configurable through the WordPress widget settings.

== Installation ==

1. Upload the `simple-most-commented-post` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to `Appearance > Widgets` and add the "Simple Most Commented Post" widget to your sidebar.
4. Configure the widget settings, including the title, number of posts, and excerpt length.

== Frequently Asked Questions ==

= Can I customize the number of posts displayed? =

Yes, you can set the number of posts to display in the widget settings.

= Can I adjust the excerpt length? =

Yes, the excerpt length can be adjusted through the widget settings.

= What if I don't want a title for the widget? =

You can leave the title field blank in the widget settings if you don't want a title to be displayed.
